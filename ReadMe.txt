Структура проекта

1)maket -┐ (папка для хранения html-макетов блоков страницы (header, footer и т.п))
	 |
         |-- footer	
	 |-- header
	 |-- news
	 |-- top-news
	 |-- personal-area
	 |-- login
	 |-- reg
	 └-- sass (из этой папки sass файлы компилируются в css в указанную папку (настраивать в gulpfile.js))
	

2)node_modules (лучше не трогать)

3)public (в этой папке хранятся media-файлы и файлы библиотек/фреймворков для их подключения + index.html для react'а) 

4)src (папка с react-компонентами и другими нужными для react'а файлами)

5)Структура (папка для psd-макетов страниц (только главных элементов,без особого дизайна))

6)Дизайн (папка для psd-макетов страниц с окончательным дизайном)
